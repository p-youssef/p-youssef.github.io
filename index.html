About Me

Dafaa Website

The site was built using the Django framework (Python) for the backend, React.js and REST APIs for the frontend, and designed using Figma. The development—both back-end and front-end—was handled by Philip youssef, with UI/UX design by Jozian Sriek.

The platform consists of two primary sections:

Client View: A user-friendly interface that streamlines navigation, ordering, and purchasing, utilizing React and REST APIs for a smooth and efficient experience.
Management View: Accessible to the Dafaa team for managing production, shipping, and financial operations with ease.
Deployed on AWS, the site ensures scalability and reliable performance.

Explore more at dafaa.it !

DeAgri website

DeAgri application aims to provide advanced solutions to farmers by accurately and quickly diagnosing plant diseases, which contributes to improving agricultural production and reducing losses. The app also includes more revolutionary features to support farmers.

We developed a website using React and Django, with a REST API, where users can download the DeAgri app and read the latest agriculture-related news and articles. The development is led by Philip youssef and Aziz Ali, with design by Jozian Sriek
